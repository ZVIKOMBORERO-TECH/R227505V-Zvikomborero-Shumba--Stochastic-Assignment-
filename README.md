# HASTS416/RM Markov Chain Assignment

## Overview
This repository contains solutions for the Markov Chain assignment with three parts:
- A1: 5-State Markov Chain Analysis
- A2: 7-State Markov Chain Analysis  
- A3: Non-Homogeneous Traffic Model

## Files
- `assignment_markov_analysis.R` - Main analysis code with complete implementations
- `run_assignment.R` - Script to execute the analysis
- `A1_*.png` - Generated visualizations for Assignment 1
- `A2_*.png` - Generated visualizations for Assignment 2

## Usage
To run the complete analysis:
```r
source("run_assignment.R")
```

## Requirements
- R (version 4.0+)
- Required packages: base R packages only

## Assignment Structure
The analysis includes:
1. **Shared utility functions** for matrix operations and Markov chain simulation
2. **Assignment 1**: 5-state Markov chain with convergence analysis and trajectory visualization
3. **Assignment 2**: 7-state Markov chain with similar analysis
4. **Assignment 3**: Non-homogeneous traffic flow model

## Output Files
- Convergence plots showing marginal probabilities over time
- State transition diagrams
- Multiple trajectory simulations
